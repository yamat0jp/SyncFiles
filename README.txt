【インストール・アンインストール】

解凍してご自由に配置していただければ使用できます。
アンインストールは本体ファイルとREADME.txt、後から作成される
初期設定ファイルのSyncFiles.txtを削除してください。


【ソフトウェア概要】

このアプリケーションはDelphi開発用に用意されたものです。
主にWebBrokerでの開発作業時にDelphiの配置機能に頼らず
ディレクトリ内部のファイルたちを上書きコピーさせます。
これにより手作業でのコピーが必要なくなり同期させやすくなります。
サイズが４MBを超えるファイルはコピーしません。
